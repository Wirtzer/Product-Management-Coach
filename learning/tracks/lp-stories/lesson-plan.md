# Leadership Principle Stories — Lesson Plan

**Updated:** 2026-01-01

## Current Status
New learner. No sessions completed yet. Begin with diagnostic to establish baseline.

## What to Cover First
1. **Story inventory** — Identify 3-5 career stories that map to common leadership principles (ownership, customer obsession, bias for action). Focus on stories with measurable outcomes.
2. **STAR structure fundamentals** — Learn the Situation-Task-Action-Result framework and practice applying it to one strong story.
3. **Quantifying impact** — Practice embedding concrete metrics and numbers into the Result section of each story.

## Recommended First Exercise
Pick one career accomplishment you're proud of. Tell it in 2 minutes using STAR format. The coach will listen, then give feedback on structure, specificity, and impact clarity. No pressure on delivery yet — this is about getting the raw material organized.

## Phase Progression
- Introduction → Structuring → Practice → Refinement → Performance
- Current phase: Introduction (identify raw material, projects, decisions, conflicts, results)

## Key Dimensions
- Star structure, content accuracy, delivery clarity, technical LP angle, strategic frame
