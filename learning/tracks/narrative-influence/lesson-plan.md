# Narrative and Influence — Lesson Plan

**Updated:** 2026-01-01

## Current Status
New learner. No sessions completed yet. Begin with diagnostic to establish baseline.

## What to Cover First
1. **Product narrative structure** — Learn the "situation → complication → resolution" framework for persuasive product communication. Understand why narrative beats bullet points for executive buy-in.
2. **The one-pager** — Master the single most important PM document format. Learn structure: problem, proposed solution, key metrics, risks, ask. Practice writing one that can be read in 3 minutes.
3. **Stakeholder communication** — Understand how to adapt the same message for different audiences: engineers, executives, customers, partners. Same content, different framing.

## Recommended First Exercise
Write a one-pager for a product feature you care about (real or hypothetical). Keep it to one page. Include: the problem (with data), the proposed solution, success metrics, key risks, and a clear ask. The coach will review it for clarity, persuasiveness, and structure.

## Phase Progression
- Observe → Imitate → Adapt → Create → Master
- Current phase: Observe (study great product narratives from successful launches)

## Key Dimensions
- Core concepts, practical application, current research, mathematical intuition
