# Agent Architectures — Lesson Plan

**Updated:** 2026-01-01

## Current Status
New learner. No sessions completed yet. Begin with diagnostic to establish baseline.

## What to Cover First
1. **What is an AI agent?** — Understand the distinction between a model that responds and an agent that plans, acts, and observes. Start with a concrete example: an AI that can browse the web, write code, or control a robot vs. one that just generates text.
2. **Tool use** — Learn how agents extend their capabilities by calling external tools (APIs, code execution, search). Understand the "thought → action → observation" loop.
3. **Planning and reasoning** — Understand how agents decompose complex tasks into steps. Explore chain-of-thought, tree-of-thought, and ReAct patterns at an intuitive level.

## Recommended First Exercise
Think of a complex task you do at work that involves multiple steps and tools (e.g., "research a competitor, summarize findings, draft a strategy memo"). Break it into an explicit step-by-step plan. For each step, identify what tool or capability is needed. The coach will use this to explain how AI agents decompose and execute multi-step workflows.
