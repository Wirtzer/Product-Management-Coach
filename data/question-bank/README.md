# Question Bank

This directory holds your interview practice questions. PM Coach uses these for mock interview sessions, timed drills, and expert persona tests.

**This directory starts empty.** Add your own questions using the format below.

---

## Question Format

Questions are stored as JSON files, one per category. See `template.json` for the schema.

### Categories

| File | Category | Description |
|------|----------|-------------|
| `product-sense.json` | Product Sense | "How would you improve X?" / "Design a product for Y" |
| `strategy.json` | Strategy | Market entry, competitive positioning, business model |
| `behavioral.json` | Behavioral | "Tell me about a time when..." / Leadership Principles |
| `metrics.json` | Metrics & Analytics | "How would you measure success?" / Define metrics |
| `estimation.json` | Estimation | Market sizing, Fermi estimation |
| `technical.json` | Technical | System design, API tradeoffs, architecture decisions |
| `execution.json` | Execution | Prioritization, roadmap, stakeholder management |
| `custom.json` | Custom | Your own categories |

---

## Adding Questions

### Option 1: Create a JSON File
Copy `template.json` and populate it with your questions.

### Option 2: During a Coaching Session
Tell the coach: "Add this question to the bank: [your question]"
The coach will create or update the appropriate category file.

### Option 3: Manually Add One Question
Append to the appropriate category file following the schema in `template.json`.

---

## Tips

- **Source your own questions** from job postings, interview prep communities, or your own interview experiences
- **Tag questions by company** to practice company-specific sessions
- **Rate difficulty honestly** — it helps the coach calibrate the session
- **Add notes** from real interviews about what worked and what didn't
