# Lesson Plan: Growth & Experimentation

## Current Focus
Introduction to growth fundamentals through concrete examples.

## Phase: Concrete Example
Start with tangible, real-world examples before abstracting principles.

## Key Topics to Cover

### 1. Growth Loops (not funnels)
- The difference between linear funnels and compounding loops
- Example: Pinterest's SEO → Content → Users → More Content → More SEO loop
- Example: Slack's team adoption → integration → stickiness → expansion loop
- Exercise: Identify the growth loop for a product you use daily

### 2. A/B Testing Fundamentals
- What you're actually testing (hypothesis, not feature)
- Statistical significance: what it means and what it doesn't
- Sample size estimation: why running a test for "long enough" isn't sufficient
- Example: Walking through a real A/B test from hypothesis to conclusion

### 3. Funnel Analysis
- Identifying conversion funnels in a product
- Where to focus: the biggest absolute drop-off vs. the biggest percentage drop
- Segmentation: why aggregate funnels lie
- Exercise: Draw the conversion funnel for a signup flow

### 4. Experimentation Design
- One variable at a time (and why people violate this)
- Choosing the right metric (and the right guardrail metric)
- When NOT to A/B test (reversible decisions, small populations, ethical concerns)

## Suggested Session Flow (First Session)
1. **Warm-up (5 min):** What growth metrics does the learner already know?
2. **Concrete example (15 min):** Walk through a real A/B test case
3. **Concept extraction (10 min):** What principles can we extract from the example?
4. **Application (10 min):** Design an A/B test for a feature the learner cares about
5. **Wrap-up (5 min):** Key vocabulary check, assign reading
